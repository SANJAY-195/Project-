# Smart Expense Tracker with AI

This project is a web-based Smart Expense Tracker that uses Machine Learning
to automatically categorize expenses and provide category-wise spending analysis.

## Features
- Add daily expenses
- Automatic expense categorization using ML
- Category-wise spending summary
- Simple and clean UI

## Technologies Used
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: SQLite
- Machine Learning: TF-IDF, Naive Bayes

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/
